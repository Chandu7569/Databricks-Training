# SQL NULL Functions Practice – README

This repository contains SQL practice queries focused on handling NULL values using different SQL functions and conditions. The queries cover real-world database scenarios using NULL handling techniques.

## Topics Covered

- IS NULL
- IS NOT NULL
- ISNULL()
- COALESCE()
- NULLIF()
- NULL handling in calculations
- Real-time business scenarios

## Query Levels

### Level 1 – Basic NULL Handling
- Finding NULL values
- Checking NOT NULL records
- Counting NULL records

### Level 2 – ISNULL()
- Replacing NULL values with default values
- Handling missing salary, bonus, stock, and amount values

### Level 3 – COALESCE()
- Returning first non-NULL value
- Multiple fallback values
- Income and payment calculations

### Level 4 – NULLIF()
- Converting values to NULL
- Avoiding divide-by-zero errors
- Conditional NULL replacement

### Level 5 – Real-Time Scenarios
- Employee earnings calculation
- Identifying incomplete records
- Product and order validation queries

### Level 6 – Advanced NULL Handling
- Final payable amount calculations
- Combined NULL handling techniques
- Filtering records using NULL conditions
