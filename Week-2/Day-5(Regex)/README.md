# SQL REGEXP_SUBSTR Practice

This repository contains SQL practice queries focused on string pattern matching and extraction using `REGEXP_SUBSTR()`. The queries demonstrate how to extract numbers, text, domains, country codes, and custom patterns from different string formats.

## Topics Covered

- REGEXP_SUBSTR()
- Pattern Matching
- Character Extraction
- Numeric Extraction
- Email Parsing
- Phone Number Parsing
- String Manipulation
- Regex Quantifiers
