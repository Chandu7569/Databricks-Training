# SQL Joins Practice – README

This repository contains SQL practice queries focused on different types of joins and real-world employee management scenarios. The queries demonstrate how to retrieve, combine, and analyze data from multiple related tables using SQL joins.

## Topics Covered

- LEFT JOIN
- INNER JOIN
- SELF JOIN
- UNION
- GROUP BY with JOIN
- NULL Handling
- Employee–Manager Relationships
- Department-wise Analysis
- Project and Salary Mapping

## Practice Scenarios

- Employees with managers
- Employees and departments
- Employees and projects
- Departments without employees
- Employees without departments
- Salary details with joins
- Employee count by department
- Missing records using NULL checks

## Tables Used

- employees
- departments
- projects
- salary
- contacts

## Skills Practiced

- Writing JOIN queries
- Handling missing data
- Combining multiple tables
- Aggregation with GROUP BY
- Understanding relational database concepts

This project is useful for practicing SQL joins for:

- Database learning
