# SQL Window Functions Practice

This repository contains practice queries for commonly used SQL Window Functions using employee data.

## Topics Covered

### ROW_NUMBER()

- Assign unique row numbers
- Row numbering based on salary
- Partition row numbering by department
- Row numbering using joining date
- Alphabetical row numbering

### RANK()

- Employee ranking based on salary
- Department-wise ranking
- Ranking using joining date
- Ranking with ascending and descending order
- Alphabetical ranking

### DENSE_RANK()

- Dense ranking based on salary
- Department-wise dense ranking
- Dense ranking using joining date
- Ranking without skipped values
- Ascending and descending dense ranking

## Features

- Beginner-friendly SQL queries
- Real-time employee dataset examples
- Department-wise partition examples
- Salary-based ranking examples
- Window function practice queries

- `RANK()`
