# SQL JOIN Operations

## Overview
This project demonstrates different SQL JOIN operations using a Student Management System database.

## Concepts Used
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN
- NULL Handling
- Multi-table Queries

## Tables Used
- Students
- Courses
- Enrollments
- Instructors

## Features
- Display students and enrolled courses
- Find courses without students
- Find students without enrollments
- Show instructors and assigned courses
- Generate reports using joins

## Technologies Used
- SQL
- MySQL / PostgreSQL / SQL Server

## Conclusion
This project helps in understanding how SQL JOINs are used to retrieve related data from multiple tables in relational databases.
