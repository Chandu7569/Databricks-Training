## Objective

This repository contains a comprehensive collection of **65 SQL queries** covering essential database concepts using `Employee`, `Department`, and `Project` tables. These queries are designed for beginners and intermediate learners to practice and improve their SQL skills.

The collection includes:

- Basic `SELECT` statements
- Filtering with `WHERE`
- Pattern matching using `LIKE`
- Date functions and conditions
- Aggregate functions (`SUM`, `AVG`, `MIN`, `MAX`, `COUNT`)
- `GROUP BY` and `HAVING`
- Sorting with `ORDER BY`
- Different types of `JOIN`s
- Subqueries and nested queries
- Salary-based and department-based analysis queries
- Real-world employee management examples

## Topics Covered 📚

### Basic Queries
- Selecting all columns or specific columns

- `GROUP BY`

### Sorting

### Joins

### Subqueries
- Highest salary
- Second highest salary
- Employees earning above department average
- Departments with maximum employees/projects
